
# Portfolio Azizjon

Bu loyiha — Azizjon Norimboyevning shaxsiy portfolio vebsayti. React, Vite, va zamonaviy frontend texnologiyalari asosida yaratilgan.

## Xususiyatlar
- Responsive dizayn
- 3 ta til (uz, ru, en) uchun lokalizatsiya
- Portfolio, aloqa, haqimda sahifalari
- SEO va ijtimoiy tarmoq linklari
- Dark/Light rejim
- Testlar (Jest + React Testing Library)

## O‘rnatish

1. Repozitoriyani klonlang:
	```bash
	git clone https://github.com/NorimboyevAzizjon/Portfolio_Azizjon.git
	cd Portfolio_Azizjon
	```
2. Paketlarni o‘rnating:
	```bash
	npm install
	```

## Ishga tushirish

Lokal serverni ishga tushirish:
```bash
npm run dev
```

Build qilish:
```bash
npm run build
```

Builddan keyin preview:
```bash
npm run preview
```

## Testlarni ishga tushirish

Testlarni ishga tushirish uchun:
```bash
npm test
```
Yoki:
```bash
npm test -- --config=jest.config.cjs
```

## Loyihada ishlatilgan texnologiyalar
- React 19
- Vite
- Jest, React Testing Library
- ESLint
- CSS Modules
- React Router DOM
- i18n (custom)
